# Next Level

A responsive landing page inspired by a Figma design, built with **HTML5**, **JavaScript**, and **Tailwind CSS**. Designed with a clean, modern layout and optimized for flexibility and customization.

## 🚀 Tech Stack

- HTML5
- JavaScript
- Tailwind CSS

## ✨ Features

- Responsive design
- Modern UI based on a Figma mockup
- Utility-first styling with Tailwind CSS
- Easy to customize and extend
